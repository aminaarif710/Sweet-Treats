## 📖 About the Project

Sweet Treats is a full-stack recipe website developed using the MERN Stack. It provides users with an interactive platform to browse recipes, discover cooking tips, and enjoy a responsive design across all devices.

---

## ✨ Features

- 🍽 Browse delicious recipes
- 🔍 Search recipes
- 👤 User Registration
- 🔐 Secure User Login
- 👨‍🍳 Recipe Details Page
- 📂 Recipe Categories
- 📱 Fully Responsive Design
- 🏠 Home Page
- ℹ️ About Page
- 🛎️ Services Page
- 📞 Contact Page
- 🛠️ Admin Dashboard
- ⚡ Fast and Modern User Interface

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/aminaarif710/Sweet-Treats
```

### Go to project folder

```bash
cd sweet-treats
```

### Install frontend dependencies

```bash
cd client
npm install
```

### Install backend dependencies

```bash
cd ../server
npm install
```

### Start Backend

```bash
npm run dev
```

### Start Frontend

```bash
cd ../client
npm run dev
```
